Adult Income Prediction using Machine Learning

This project focuses on predicting whether a person earns more than 50K per year using the Adult Income dataset. The aim of this project was to work with tabular data, clean and prepare the dataset, train different machine learning models, and compare their performance.

What This Project Includes
Data cleaning and preprocessing
Exploratory data analysis
Feature encoding and scaling
Training multiple machine learning models
Model evaluation using accuracy, precision, recall, F1-score, and ROC-AUC
Artificial Neural Network implementation
Feature importance analysis
Basic unsupervised clustering
Models Used
Logistic Regression
Decision Tree
Random Forest
Gradient Boosting
Artificial Neural Network
K-Means Clustering
Tools and Libraries
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
PyTorch
What I Learned

This project helped me understand how machine learning models can be applied to structured data. I learned how to preprocess real-world datasets, compare different models, evaluate performance properly, and understand which features have the most impact on predictions.
